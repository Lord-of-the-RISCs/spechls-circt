SpecHLS.gamma (SpecHLS::GammaOp)

Illustrates how to define an operation.

The SpecHLS.foo operation illustrates how to define a new operation in a dialect. It uses an operation trait to declare that it has no side effects.

This operation takes an integer argument and returns an integer.

Example:

%0 = constant 2 : i32
// Apply the foo operation to %0
%1 = SpecHLS.foo %0 : i32

Traits: AlwaysSpeculatableImplTrait

Interfaces: ConditionallySpeculatable, NoMemoryEffect (MemoryEffectOpInterface), hasCustomSSAName

Effects: MemoryEffects::Effect{}
